# jsbridge (功能需求)

**没事找事APP `WebView`与`UIView` `jsbridge`**

*   城市
    *   得到城市地点和城市ID
    *   打开城市页面选择后回调
*   分享
    *   `js`打开`app`分享`app`分享后回调

**WebView功能实现** 
1.  浏览器`user-agent`的规范
-   打开浏览器判断域名是否属于允许的域名 如果允许种 `Cookie`
-   浏览器关闭按钮（参考微信）
-   下拉刷新（`WebView`与`app`导航分层）
    *   下拉刷新整个`WebView`
    *   下拉回调`js`进行局部刷新
-   导航选项菜单（参考天猫）
-   `alert`弹窗，`confirm`弹出， `loading`
-   **`js`进行各个页面的调用包括参数和回调**

*最后一个功能比较重要，需要整体规划页面，和页面跳转，每个页面需要有传参*
